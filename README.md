# SpringBoot+Vue的小区物业管理系统

#### 介绍
业主功能：业主信息，房屋信息，小区信息，缴费信息，公告通知，报修申请，投诉上报，车位租用
物业管理员功能：报修申请，业主管理，房屋信息管理，安保管理，绿化管理，投诉管理，库存物品管理
财务人员功能：收费管理，财务管理，欠费管理
系统管理员：用户管理，通知管理，以上角色所有功能
